# Feather-Talk (페더톡)

저사양 환경에서도 VTuber 아바타를 간단하게 표현할 수 있는 웹 기반 도구입니다. 별도의 빌드 없이 브라우저에서 바로 실행되며, 최소한의 이미지와 레이어로 자연스러운 움직임을 제공합니다.

## 주요 기능
- 리깅 프리셋: 모션에 따른 파츠 움직임을 미리 계산해두어 간단한 설정으로 동작합니다.
- 이미지 최소화: 8장의 이미지와 6겹 레이어로 캐릭터 움직임을 구현합니다.
- 가동범위: 고개 상하좌우 움직임, 눈 깜빡임, 입 벌림/닫힘 애니메이션을 지원합니다.
- 저부하 모션: 웹캠 트래킹 같은 고부하 방식 대신 자동/마우스 기반 모션을 제공합니다.
- 자동 모션: 라이브 창에 포커스가 없을 때 일정 간격으로 랜덤 움직임을 보여줍니다.
- 마우스 모션: 라이브 창 위 마우스를 따라 캐릭터가 움직이며 더 자연스러운 움직임이 가능합니다.

## 개선된 기능 (블로그 소개 기준)
- 여러 이미지를 레이어처럼 자유롭게 추가하고 Rig 타입/레이어 위치를 설정할 수 있습니다.
- 로컬 이미지 파일을 파일 버튼으로 바로 등록할 수 있습니다.
- 좌측 상단 톱니바퀴로 설정 패널을 열고 닫을 수 있습니다.
- X/Y 위치 조절 기능이 추가되었습니다.
- 자동 모션 속도를 범위 내 랜덤으로 선택하는 옵션이 추가되었습니다.
- 카메라 인식 기능이 추가되어 실제 얼굴 움직임에 맞춰 캐릭터가 반응합니다.
  - 얼굴 좌우상하 움직임, 몸 좌우 이동, 눈 깜빡임, 입 움직임을 추적합니다.
  - 카메라 연결이 필요하며, 이미지 인식 기반이라 완벽하게 정확하지 않을 수 있습니다.
- 기타 수정: 호스팅 없이 실행 시 라이브 페이지 접근 문제 수정, 이미지 경로 제거 시 기본 이미지 복원, 입력 이미지 경로 저장 처리 등등..


## 사용 방법 (코드 없이)
- https://404errrror.github.io/feathertalk/ 에서 바로 실행해볼 수 있습니다.

## 사용 방법 (코드를 통해)
1. `start-local.bat` 실행 또는 `index.html`을 실행시킵니다.
2. 레이어/이미지와 설정을 입력한 뒤 라이브 버튼을 클릭합니다.
